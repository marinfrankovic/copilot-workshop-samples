# Copilot Workshop — Sample Files

Sample files for the full-day workshop **"From idea to working script"** —
learning to use GitHub Copilot by describing what you want in plain English
(no coding background required).

Open this whole folder in VS Code (**File ▸ Open Folder**) so Copilot can see the files.

```
.
├─ lab1/
│   └─ mystery_snippet.py     ← Lab 1: ask Copilot to explain it
├─ lab2/
│   ├─ sales_january.csv      ← messy on purpose ($ signs, mixed case, commas)
│   ├─ sales_february.csv     ← different column names & date format
│   └─ product_prices.csv     ← price lookup table
└─ capstone/
    └─ START_HERE.md          ← bring your own task
```

## Labs

- **Lab 1 — Talk to your code:** ask Copilot to explain `lab1/mystery_snippet.py`,
  add comments, and rename things so a human can read it.
- **Lab 2 — Automate a business task:** clean and merge the `lab2/` sales exports,
  join the price list, and generate a simple summary report.
- **Capstone — Build your own:** bring a real task from your job (see `capstone/START_HERE.md`).

## About the data

All names, emails, and numbers are **fictitious**. People (Megan Bowen, Adele Vance,
Alex Wilber, Diego Siciliani) and the company **Contoso** are Microsoft's standard
sample identities. There is **no real personal data** in this repository.

## Tips

- No setup or installs are required to *read* these files.
- If a lab asks you to *run* a script, a coach will help — or Copilot will give you the one command to use.
- Always work on a **copy** of real data, never the original.
