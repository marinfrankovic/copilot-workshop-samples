# Capstone — Build Your Own

Bring a **real task from your own job** and build something for it with Copilot.
You don't need to know how to code — you'll describe what you want in plain English.

## Pick something small and real
Good capstone ideas (steal one if you're stuck):

- Rename or organise a folder full of files by date or keyword
- Turn a messy spreadsheet into a clean summary or chart
- Merge two exports (CSV/Excel) and find what changed
- Generate a templated email or report from a list of names/values
- Pull a few numbers from a file and produce a one-line summary
- Reformat a list (e.g. CSV ➜ nicely formatted text, or JSON ➜ table)

## The loop you'll use
1. **Describe** the goal in one or two plain sentences.
2. **Generate** — let Copilot draft it.
3. **Run** it on a COPY of your data (never the original).
4. **Refine** — tell Copilot what's wrong or what to change.
5. **Repeat** until it does the job.

## Safety rules
- Work on a **copy** of any real file.
- Read what the script does before you run it (ask Copilot to explain it).
- If you're unsure, ask a coach.

## When you're done
Be ready to show (optional): what you asked for, and what you got.
